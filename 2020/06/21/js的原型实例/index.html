一些概念

函数的封装、多态、继承：

封装：

// 把数据和操作这组数据的函数放在一起，称为对象。此过程叫封装

// 封装的另一层含义： 将复杂的操作或计算写成函数， 调用时只需考虑高晨概念，进而无需考虑该操作的运作细节

多态：

// 多个不同类型的对象具有相同的一组接口（方法与方法接受的参数类型与返回值的类型）

继承：

// 当一个类型拥有另一个类型的所有特性。可以让这一个类型直接通过某种方式直接获得该类型的所有属性和方法， 即称为继承

// 被继承称为父类，继承者称为子类

原型

定义：

每个对象除了拥有自己的属性以外，几乎都包含一个原型（prototype）。原型是另一个对象，是对象的一个属性来源。当访问一个对象不包含的属性时，就会从对象的原型中搜索属性，接着是原型的原型依次类推。
