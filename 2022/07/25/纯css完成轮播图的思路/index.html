纯css实现轮播图思路

发表于 2022-07-25 分类于 CSS

前言

本文探讨的是如何在纯css环境中完成轮播图的编写

第一种方案

将轮播图排成一排然后通过动画效果实现轮播

HTML部分：

CSS实现步骤：

  • 父子元素需要定位，子元素相对定位
  • white-space: nowrap;强制图片不换行
  • 父元素overflow: hidden;隐藏多余的图片
  • font-size: 0 图片为行内元素，之间会有空隙，通过设置字号为0，消除间隙
  • 动画效果的实现：
    • 轮播图画面是每个一段时间改变一次，那么对应着图片每隔一段时间向左移动一次然后停留一会
    • 当最后一张图片移动到最左边时结束，并回到初始状态

第二种方案

通过input+label的方式实现图片的变换

HTML部分：

CSS 实现步骤：

  • 父元素overflow: hidden;并定位
  • input元素定位到父元素底部中间
  • img绝对定位到left: 100%;
  • label定位到两侧中间
  • input与ing，左标签、右标签关联
    • 图片left: 0; z-index: 15;
    • 左右标签z-index: 20

第三种方案

在第二种的基础上添加缓动和动画效果实现轮播特效

实现思路：

  • 为未被选择的图片添加一个离场动画

  • 同时为被选择的图片取消这个动画并添加一个入场延时
