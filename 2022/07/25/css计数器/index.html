CSS计数器

发表于 2022-07-25 分类于 CSS

前言

本博客主要是为了理解和运用css计数器

css计数器的概念及定义

本质上CSS计数器是由CSS维护的变量，这些变量可能根据CSS规则增加以跟踪使用次数。这允许你根据文档位置来调整内容表现

计数器的值通过使用counter-reset 和 counter-increment 操作，在 content 上应用 counter() 或 counters()函数来显示在页面上

使用计数器

要使用计数器之前必需先重置一个值。默认为0，使用counter()函数给元素增加计数器

重置counter-reset（实际起命名的作用）

增加计数counter-increment

显示计数伪元素中 content : counter()

举例

计数器的延伸部分

  • counter-reset 的命名规则

counter-reset: name1 num1 name2 num2

可以看出可以实现多个计数器同时命名 num可以为正，负，0；

也可以为inherit或none继承

  • counter-increment的计数规则

    默认变化值为1后跟随数字表示每次的变化值（可以为负）

    当增计数用于伪元素自身会是什么样的呢？

会发现当counter-increment用于父元素和伪元素自身时，都会产生计数增加的效果

  • content 的counter()参数

content: counter(name, sytle)

将计数器以罗马字符显示

  • css计数器在display: none时计数不增加
  • css计数器在IE6或IE7上不支持

counter的嵌套

counters()加了个s实现了嵌套的的功能

基本使用方法：

counters(name, string);或counters(name, string, style)

嵌套的核心在于通过子元素对父元素计数器的counter-reset重置（相当于重新命名）

string 为一组多个counter之间的连接符
